História da Computação

História dos computadores e da internet, inicialmente sempre esteve ligada a guerra, na corrida da comunicação a guerra foi uma grande aliada, no desenvolvimento de novas tecnologias, sendo Estados unidos e União soviética os maiores responsáveis pelo avanço durante a 2ª guerra mundial.

Tim Berners-Lee, o criador do 1º navegador o World Wide Web w.w.w, e também foi o criar dor da linguagem de marcação HTML, que foi um modo de padronizar como os sites seria construídos, para que as coisas não saíssem do controle, já que cada instituição estava criando um modo de se comunicar internamente.

Client-side e Server-side

Client-side

Cliente-side, não é nada mais do que o LADO DO CLIENTE, tudo que é executado levando em consideração o equipamento do cliente, é considerado client-side, na programação temos programas específicos que rodam em determinada tecnologia, como por exemplo para rodar o HTML e CSS do lado do cliente usamos o navegador que interpreta e traz uma interface gráfica e amigável ao usuário. Ainda temos a linguagem de programação, JavaScript, que roda tanto lado cliente quanto do lado do servidor, entretanto o modo que essa tecnologia é interpretada em ambos os lados é diferente, mesmo que usem o mesmo interpretador o V8-ENGINE, no lado cliente quem usa esse interpretador é o navegador.

Server-side

Server-side, significa LADO DO SERVIDOR, é tudo que é servido ao cliente, toda APLICAÇÃO WEB, precisa ser armazenada em algum lugar, isso ocorre no servidor, que tem a função de literalmente servir o cliente com a aplicação solicitada, exemplo disso é quando acessamo o Google, o site do Google está armazenado no servidor do Google. Falando de programação do lado do servidor, usamos basicamente linguagens diferentes para manipular o servidor, as chamadas linguagens BACK-END, de modo geral temos varia linguagens que são usadas, como,PHP, JS, C#, Java Ruby, entre outras, entretanto a mais utilizada atualmente (no ano de 2023), é o JavaScript, que utiliza o Node JS, para rodar o interpretador V8-ENGINE.

Comunicação

De modo geral a comunicação entre Cliente e Servidor, utiliza a internet para que ocorra. sendo necessário uma requisição do lado cliente.

Requisição do cliente servidor

Cliente acessá o navegador, digita o endereço correspondente ao site que deseja, o navegador direciona para servidor, a informação é processada pelo servidor, que envia uma copia da informação para o navegar em forma de texto, que por sua vez interpreta a informação, e converte em modo gráfico.

Estrutura básica do HTML

A estrutura básica do HTML é dividida em três partes, sendo elas as tags "html", "head" e "body".

Tag html

Usado como um contentor que organiza de forma clara onde é o inicio e o fim da do código.

Tag head

A tag head, é a cabeça do codigo, é onde se encontra as configurações e conexões que o documento vai receber.

Tag body

A tag body, é o corpo do site, onde se encontra toda informação visível ao usuário final.

Lista Ordenada

Como o proprio nome já diz, lista ordelada é um lista organizada em orde numérica, sendo que o navegador interpreta a ordem do jeito que é inserido.

Lista não Ordenada

Diferente da lista ordenada, a lista não ordenada traz uma no inicio de cada item um caractere, mas de mesmo modo que a ordenada, a ordem dos itens é expressada da mesma maneira na ordem que é colocada.

Tag li

A Teg utilizada neste caso é ol, que informa ao navegador que os itens entre a tag ol de abertura e a teg ol de fechamento é uma lista ordenada. Cada item dentro da lista é contido por uma teg de item que é representado li.
